# JUEGO DE AHORCADO EN JAVA
Creo un sencillo juego de consola tipo ahorcado en Java, donde el jugador debe ingresar de forma individual cada letra y tiene un maximo de intentos en caso de fallar.

La idea del juego es el jugador adivine todas las letras que corresponden a la palabra. Si lo hace, gana; caso contrario, tiene un maximo de intentos (10 en total) en caso de que la letra ingresada no coincida con alguna de las letras de la palabra designada.
